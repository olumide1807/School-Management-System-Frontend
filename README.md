# School Flow

**School Flow** is an innovative application designed to streamline and enhance the management of student data, admissions, and school-related processes. This project aims to provide a comprehensive solution for school administrators to manage student information efficiently, with features that facilitate data entry, review, and editing.

## Features

- **Student Data Management**: Easily manage student profiles, including personal information, contact details, and academic records.
- **Guardian Information**: Add and manage guardian details with options to switch between manual entry and selection from predefined lists.
- **Dynamic Form Handling**: A flexible form system that adjusts based on user interactions, such as toggling between manual and selection modes.
- **Responsive Design**: The application is designed to be fully responsive, ensuring usability across different devices and screen sizes.
- **Multi-Step Form**: Use a step-by-step approach to edit and review student data, making the process intuitive and organized.
- **Preview and Review**: Review changes before final submission, ensuring data accuracy and completeness.
